If you use VC++ 5.0 and build your driver in the IDE environment, you will
get this error ! Use the LINKER in your NTDDK 5.0 , add the path
$(DDKROOT)\bin in front of the old path where DDKROOT is the directory you
install your NTDDK 5.0
